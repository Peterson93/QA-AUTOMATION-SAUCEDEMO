# QA Automation Project - SauceDemo

Automatización de pruebas end-to-end usando Python, Selenium y pytest sobre la aplicación SauceDemo.

## Tecnologías
- Python
- Selenium
- pytest

## Casos de prueba implementados
- Login válido
- Login inválido
- Agregar producto al carrito
- Validación de producto en carrito

## Evidencia
El proyecto incluye captura automática de pantalla cuando ocurre un fallo.

## Ejecución
PowerShell
pytest
